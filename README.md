# AI Assisted Creations

> 1% ideas. 99% bad prompts.

A growing collection of tools, experiments and games built with [Claude](https://claude.ai) by K Williamson.

## Browse the collection

**👉 [kiatinw.github.io/studious-creations](https://kiatinw.github.io/studious-creations)**

The live site is the source of truth — all projects are listed there.

## What's in here

Everything is built with Claude as the primary development tool. Projects range from classroom utilities to revision games to data tools — mostly for a secondary school maths context.

New projects are added whenever something is worth shipping. Check the site.

## Stack

- Vanilla HTML / CSS / JS
- No frameworks, no build steps
- Hosted on GitHub Pages
