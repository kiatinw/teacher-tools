Importing Data

This tool can import your faculty's structure from a CSV file or by pasting data from a spreadsheet (like Excel or Google Sheets). This is the fastest way to get set up.

From Sentral: You can often generate a report of all classes for your faculty. Look for an option to export this as a CSV or copy it to Excel. The key is to get a list with at least these three columns: Course Name, Class Code, and Teacher Name.

The import requires the following columns with these exact headers:

  • course_name
  • class_name
  • teacher_name
  • periods (optional, defaults to 5)
  • markers_needed (optional, defaults to 2)

You can use the "Download CSV Template" button to get a file with the correct headers.
